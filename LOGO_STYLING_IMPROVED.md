## 📐 Technical Details

### **Sidebar Logo:**
```css
Container:
- Background: white
- Border radius: 15px
- Padding: 15px
- Max width: 180px
- Box shadow: subtle

Logo:
- Max height: 80px
- Object-fit: contain
- Width: 100%
- Auto height
```

### **Login Page Logo:**
```css
Container:
- Background: white
- Border radius: 20px
- Padding: 20px
- Max width: 250px
- Box shadow: prominent

Logo:
- Max height: 100px
- Object-fit: contain
- Width: 100%
- Auto height
```
